# コイカツ セーブデータ編集ツール

以下のことができます。

* キャラクタの名前の変更
* キャラクタの性格の変更
* キャラクタの性感帯の変更
* キャラクタの属性の書き換え
* キャラクタの置き換え

キャラクタの置き換えは、置き換えるキャラのPNGファイルを指定します。
なお、好感度などはそのまま引き継がれます。


## 実行方法

gui.exe をセーブデータを指定して実行するか、gui.exe にセーブデータをドロップしてください。

```
> gui.exe file01.dat
```

exeではなく python のコードを実行する場合には、python 3.6 以降が必要です。

```
> python gui.py file01.dat
```

セーブデータを書き換えた場合、もとのファイルは拡張子 .old を付け加えてコピーされます。

下着ランダムチェンジャ機能
--------------------------------------------------------------------------------
■■これは
・セーブデータをいじって、女性キャラの下着をランダムに変更する機能です。
・下着のコーディネイトは、あらかじめ設定する必要があります。


■■つかいかた
1) セーブデータをつくる
2) KKUCを起動
3) いじるセーブデータを選ぶ
4) 下着データを蓄積する
5) チェンジ！


■■１）セーブデータをつくる
・とにかくコイカツをプレイし、セーブデータを作ってください。


■■２）KKUCを起動を起動
・gui.exeをダブルクリック。


■■３）いじるセーブデータを選ぶ
・ダイアログが出ます。いじくりまわしたいセーブデータを選んでください。


■■４）下着データを蓄積する
・キャラごとに操作パネルが出ます（KoikatuSaveDataEditの機能です）。
・パネルの下側に「〇〇の下着を記録」というボタンがあります。
・これを押すと、そのキャラの「コーディネイトタイプ：学生服（校内）」での「ブラ」「ショーツ」を記録し、テキストファイルにしてKKUCディレクトリ内に格納します。
・この機能で記録した下着データが、ランダムチェンジされて出てきます。
・つまり。
　　・コイカツを普通にプレイ
　　・夜時間の座席表からキャラ編集し、お好みの下着をコーディネイト
　　・セーブ
　　・KoikatuSaveDataEditUCを起動し、下着を着せたキャラの記録ボタンを押す
　　・これを繰り返すと、下着データがたまっていく


■■５）チェンジ！
・「Save&Quit(Change)」をクリックすると、キャラのパラメータ操作（KoikatuSaveDataEditの機能です）とともに下着データがランダムに選ばれ、チェンジします。
・この時点ではセーブデータをいじっただけですので、ゲームからロードして反映させること。


■■ご注意
・下着データのファイル名は変えてもだいじょうぶです（拡張子は変えないように）。
・ファイル名は中身が見てわかる名前に変更することをおすすめします。

下着ランダムチェンジャのキャラ別設定機能
--------------------------------------------------------------------------------
■■これは
・下着ランダムチェンジャに追加した機能です。
・キャライメージに合わせて下着を選んでほしい、という欲望を叶える機能です。
・キャラごとに設定ファイルを用意し、それを編集することで、着用する下着を選べます。



■■つかいかた
1) セーブデータをつくる
2) KoikatuSaveDataEditUCを起動
3) 下着設定データを編集


■■１、２）
まあとりあえず。


■■３）下着設定データを編集
・セーブデータを選択した時点で、登校中のキャラの下着設定データが出来ています（_equip_おなまえ.ini）。
・テキストファイルで開きます。
・ざっと眺めると、７つに別れたエリアのそれぞれに、下着データのファイル名が書かれています。
・このエリアは、コーディネイトの分類です。頭についた番号は、以下を表しています。
	０：学生服（校内）
	１：学生服（下校）
	２：体操服
	３：水着
	４：部活
	５：私服
	６：お泊り
・例えばエリア２内の「白下着.txt」は、「体操服の時に白下着を選ぶかどうか」の設定、ということに。
・下着データのファイル名の横、1はその下着を選ぶ、0は選ばない、という意味です。ここをお好きに変更し、保存してください。


■■ご注意
・「この機能を使わない」ことはできません。すいません。
・下着設定データのファイル名は、基本変えないように。
・名前をキャラ名に合わせれば、使い回しができます。
・下着を追加すると、自動で下着設定データにも追加されます。
・下着設定データを新規追加した際、全員水着時の設定はすべて0（選ばない）になっています。
・また下着データを追加した際は、全員水着と部活の設定はすべて0（選ばない）になっています。部活によって着るものがちがうため、やむなくそうしています。
・…なので、デフォルトでは、水着以外はすべて下着を変えるようになっています。水泳部あたりがあやしくなりますので、お早めの変更をおすすめ。

試着機能
--------------------------------------------------------------------------------
■■これは
・一旦作った下着データの中身を確かめる機能です。


■■つかいかた
1) 下着データをつくる
2) 試着ボタンを押す
3) セーブデータをいじる
4) 確かめる


■■１）下着データをつくる
割愛。


■■２）下着データをつくる
・キャラごとに操作パネルが出ます（KoikatuSaveDataEditの機能です）。
・パネルの下側に「〇〇に下着を試着」というボタンがあります。
・押すと下着データを選ぶダイアログが出てくるので、選択します。
・と、そのキャラの「学生服（校内）」時の「ブラ」「ショーツ」が、選択した下着になります。


■■３）セーブデータをいじる
・「Save & Quit(Change)」ボタンを押すと、セーブデータが書き換えられます。
・ランダムチェンジャも同時に起動しますが、試着したキャラはランダムにならず、選択した下着を着用します。


■■４）確かめる
・この時点ではセーブデータをいじっただけですので、ゲームからロードして反映させること。
・座席表などから確認してください。


■■ご注意
・試着だけする、ということはできません。必ずランダムチェンジャも起動します。
・試着した事自体は記録されませんので、KKUCを起動する度に試着してください。
（下着を固定するための機能ではない）
